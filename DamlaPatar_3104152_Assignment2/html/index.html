Semantic HTML

Semantic HTML refers to the practice of using HTML tags and attributes to represent the significance and organization of information on a website, rather than solely focusing on its aesthetic presentation. In other words, it involves using HTML in a way that accurately communicates the purpose and context of each piece of content on a webpage.

Flex Box

Flexbox is a type of CSS layout that enables the flexible positioning of items within a container, resulting in layouts that can adapt and adjust to various screen sizes and device types.

Grid Layout

The Grid is a layout model in CSS that allows the creation of layouts based on a two-dimensional grid, providing precise management of columns, rows, and the positioning of elements.
